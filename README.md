### Usage

Install package:

```sh
$ npm install --save-dev @commitlint/cli @lukaszbieniek/commitlint-config
```

Extend your config file:

```js
// commitlint.config.js
module.exports = {
  extends: ["@lukaszbieniek/commitlint-config"],
};
```
